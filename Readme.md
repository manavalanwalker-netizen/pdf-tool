# PDF Tool (Basic)

A simple desktop tool for Windows to **merge** and **split PDFs**.  
Built with Python + Tkinter + PyPDF2.  

---

## ✨ Features
- Merge multiple PDF files
- Split PDF by page range

---

## ▶️ Run with Python
1. Install Python 3.10+
2. Install dependencies:
   ```bash
   pip install -r requirements.txt
